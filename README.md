## 📄 What is ssbu-less-lag?

ssbu-less-lag is a mod for Super Smash Bros. Ultimate. It removes 2 frames of input delay in the game. By using this mod, you can experience a more responsive gameplay. This version is archived, meaning it is no longer updated but still works as expected.

This tool is designed for players who want smoother controls in SSBU without complex setup.

## 💻 System Requirements

To run ssbu-less-lag, your computer needs to meet these requirements:

## 🔧 How It Works

The mod reduces input delay by adjusting the game’s frame processing. This makes controls feel faster.

Because it removes 2 frames of delay, your inputs happen closer to when you press the buttons. This helps especially with competitive play.
